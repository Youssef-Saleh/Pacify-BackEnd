(static) put/addSongsToAlbum(req, _id, songName, _id, res)

This function would request info of the user ID and requirs that he is an artist. it will also request the song name and the album ID. It searches the collection of users and finds the particular artist that wants to add the song. Then it finds the particualr album that the artist want to add the song inside it. It would add this song to the specified album and update the info of the artist.
Parameters:
Name Type Description
req * requesting the needed info from postman as the following:
_id * the user's id generated by the mongoBD as requested from the postman
songName string is the name of the song as requested from the postman
_id * the album's id generated by the mongoDB as requested from the postman
res string it tells the artist either the song was added successfully or an error occured
