(inner) post/createAlbum(req, _id, nAlbum, imagePath, name, year, uploadedAlbums, res)

This function creates an album and adds it in the database by requiring the ID of the user and that the user is already an artist. It also requires all the important info of the album. Then it updates the user info and adds the album in the database.
Parameters:
Name Type Description
req * requesting the needed info from postman as the following:
_id * the user's id generated by the mongoBD as requested from the postman
nAlbum Object the new album created.
imagePath string the link of the image of the album
name string the name of the album
year Number the year of releasing the album
uploadedAlbums string requires the name of the album to be added int the array of uploadedAlbums
res string it tells the artist either the album was created successfully or an error occured
